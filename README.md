# ✨ ParticleText

A real-time interactive particle text and sphere visualization built using HTML, Canvas, and JavaScript.

---

## 🚀 Features

- 🌐 Dynamic particle sphere animation  
- 🔤 Text to particle transformation  
- 🎨 Color changing particles  
- ✨ Smooth animations  
- 💥 Interactive effects  

---

## 🎮 How to Use

1. Open the project in browser  
2. Type any text  
3. Watch particles form your text  

---

## 🛠 Tech Stack

- HTML  
- CSS  
- JavaScript  
- Canvas API
